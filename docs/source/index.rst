Welcome to MERLIN's documentation!
=================================

This vignette provides an introduction to the MERLIN package. R package MERLIN implements MERLIN for causal heterogeneity using summary statistics.

Install the development version of MERLIN by using the ‘devtools’ package. Note that MERLIN depends on the ‘Rcpp’ and ‘RcppArmadillo’ packages, which also require appropriate settings of Rtools and Xcode for Windows and Mac OS/X, respectively.
